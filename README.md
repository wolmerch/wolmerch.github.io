# Wolmerch - открытый код на GitHub (RU)

В этом репозитории вы можете открыть и использовать открытый код от интернет-магазина Wolmerch, но использование имени Wolmerch и стиля, похожего или отсылающего на официальный стиль магазина, запрещено. Все права, а также данные защищены.

## Что реализовано на сайте
#### 1. Мини-игра
    Игра, позволяющая получать промокоды на скидку от 100₽ до 400₽ и представляющая собой квадрат 4 * 4,
    и кирки из игры Minecraft, которые ломают блоки, а также "рулетка" которая выбирает сколько раз 
    можно использовать промокод.

#### 2. Корзина
    В которой вы можете использовать промокоды, которые вы получили в мини-игре, а также заполнять данные о 
    себе: имя, телефон и адрес (для доставки, все данные отправляются в Telegram администратору).

#### 3. Каталог, избранное, хранилище промокодов и раздел "О нас"
    В каталоге находятся товары, которые вы можете добавлять в Избранное. Собственно, все товары 
    помеченные ❤ хранятся именно там. В хранилище промокодов вы можете найти все свои промокоды, 
    полученные в мини-игре, а в разделе "О нас" вы сможете узнать больше про нашу команду.
    
## Как использовать
Вы можете начать пользоваться сайтом локально, т.е. скопировать все коды и разместить их в свой папке. 
Также есть второй вариант: зайти на [сайт Wolmerch](https://wolmerch.github.io).

# Wolmerch - open source on GitHub (EN)

In this repository, you can open and use the open source code from the online store Wolmerch, but the use of the name Wolmerch and a style similar to or referring to the official style of the store is prohibited. All rights and data are protected.

## What is implemented on the website
#### 1. A mini-game
    A game that allows you to receive promo codes for discounts from 100₽ to 400₽ and is a 4 * 4 square, 
    and picks from Minecraft games that break blocks, as well as roulette, which selects how many times 
    you can use a promo code.

#### 2. Сart
    In which you can use the promo codes that you received in the mini-game, as well as fill in information 
    about yourself: name, phone number and address (for delivery, all data is sent to the Telegram 
    administrator).

#### 3. Catalog, favorites, promo code storage, and the "About Us" section
    The catalog contains products that you can add to your Favorites. Actually, all products marked with ❤ 
    are stored exactly there. In the promo code repository you can find all your promo codes received in the 
    mini-game, and in the "About us" section you can learn more about our team.

## How to use
You can start using the site locally, i.e. copy all the codes and place them in your folder. There is also 
a second option: go to [website Wolmerch](https://wolmerch.github.io)
